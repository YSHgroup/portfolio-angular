Programador FullStack @ Moebius

Abril 2020 - Actualidad

Encargado del levantamiento de una aplicación web para gestión de datos turísticos. Las tecnologías bases involucradas en el desarrollo incluyen Laravel (API REST) y Angular en su versión 9. Mis tareas principales traducían las ideas del proyecto en interfaces funcionales, limpias y modernas. Siempre encantado y apasionado desde el primer momento en todo el proceso de desarrollo

Analista de sistema @ Proyex

Enero 2019 - Octubre 2019

Mis tareas principales abarcaron el desarrollo de una aplicación web realizada en Laravel, cuya función principal consistía en procesar datos de consumo de aguas industriales que se almacenaban en bases de datos para luego presentar informes, gráficos y alertas con alarmas automatizadas. Este proceso de monitoreo de datos y bombas, era facilitado por un panel de geolocalizacion que estuve programando en OpenLayers. Otras tecnologías usadas en los requerimientos fueron blade, webpack, bootstrap y javascript.

Un trabajo que disfrute de principio a fin.
